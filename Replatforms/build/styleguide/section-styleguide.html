Grid

Source: pages/_styleguide.scss, line 58

Grid framework based on Bootstrap, which follows stand alone grid setup provided by Bootstrap https://github.com/twbs/bootstrap/blob/v4-dev/scss/bootstrap-grid.scss. we made some changes for project scope:

  • class "row" changed to "o-grid"
  • class "col-*" changed to "o-grid__item"
  • clean up of bootstrap variables.scss for having grid variables only
  • NOTE: usage similar to bootstrap with sm, md, lg modifier
